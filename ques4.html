List of HTML Tags

Below is an ordered list of common HTML tags along with their descriptions:

  1. html: The root element of an HTML document.
  2. head: Contains meta-information about the document.
  3. title: Defines the page title.
  4. meta: Provides metadata about the document.
  5. link: Links external resources like stylesheets.
  6. script: Embeds or references JavaScript code.
  7. body: Encloses the visible content of the web page.
  8. h1 - h6: Define hierarchical headings.
  9. p: Defines paragraphs of text.
  10. a: Creates hyperlinks.
  11. img: Embeds images.
  12. ul: Defines an unordered (bulleted) list.
  13. ol: Defines an ordered (numbered) list.
  14. li: Defines list items within <ul> and <ol> lists.
  15. table: Creates tables for data.
  16. tr: Defines table rows.
  17. th: Defines table header cells.
  18. td: Defines table data cells.
  19. form: Creates interactive forms.
  20. input: Creates input fields in forms.
